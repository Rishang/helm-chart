## `serviceaccount`

Create a Kubernetes `ServiceAccount` for this release.

Rendered by: `templates/service-account.yaml`

```yaml
serviceaccount:                 # struct | Optional ServiceAccount to create
  name: my-serviceaccount       # string | Required (template expects `.Values.serviceaccount.name`)
  labels: {}                    # map    | Optional additional labels
  annotations: {}               # map    | Optional additional annotations
```
