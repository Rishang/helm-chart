# Common Helm Charts

A collection of general-purpose, highly configurable Helm charts for deploying Kubernetes applications in a convenient way.

## Charts

- [component-chart](helm/component-chart): A general-purpose Helm chart for deploying application components with a unified values.yaml.
- [loki-stack](helm/loki-stack): A Helm chart for deploying a Loki stack with Prometheus and Grafana.
